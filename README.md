[o]  **1. 주어진 문제를 해결하는 완성된 코드가 제출되었나요?**
- 루브릭3까지 좋은 아이디어를 적용해 코드를 완성함
- 출력까지 깔끔하고 가독성 좋게 체계적으로 코드를 짠 것을 느낌
- 아이디어에서 더 디테일한 부분까지 설계하여 코드를 구성함(세금을 떼는 부분까지..!!o0o)
  
[o]  **2. 전체 코드에서 가장 핵심적이거나 가장 복잡하고 이해하기 어려운 부분에 작성된 
	주석 또는 doc string을 보고 해당 코드가 잘 이해되었나요?**
- 각 메소드마다 어떤 역할을 수행하는 지 설명 다 적어둠
- 각 메소드, 코드 등이 어떻게 수행하는 지 잘 설명해주셔서 본인도 이해를 하고 코드를 짠 것 같고, 리뷰어도 그 설명을 듣고 쉽게 코드의 흐름을 이해함

[o]  **3. 에러가 난 부분을 디버깅하여 문제를 “해결한 기록"을 남겼거나 "새로운 시도 
또는 추가 실험"을 수행해봤나요?**
- 에러가 났던 부분에서 어떤 과정으로 에러를 고쳐나갔는지를 잘 설명해줌, 그만큼 본인의 코드를 잘 이해하고 있음
        
[o]  **4. 회고를 잘 작성했나요?**
- error가 났던 부분과 코드를 작성하면서 도중에 생각났던 디테일한 부분을 수정했던 과정을 잘 이야기함

[o]  **5. 코드가 간결하고 효율적인가요?**
